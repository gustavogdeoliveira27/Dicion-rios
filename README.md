# Dicionários
Exercícios de dicionários propostos na matéria de Raciocínio Algorítmico na faculdade PUCPR. Os exercícios usam a linguagem de programação Python, e foram desenvolvidos no leitor de código-fonte VS Code. Utilizam diversos métodos, como dict(), clear(), copy(), fromkeys(), get(), items(), keys(), values(), pop(), popitem(), update(), entre outros.
